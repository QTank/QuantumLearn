Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure of a webpage using elements like headings, paragraphs, and links.

  • Heading: Defines the titles or subtitles.
  • Paragraph: Contains text content.
  • Link: Navigates to other pages or resources.

HTML is essential for building the web. Start practicing by writing simple HTML code!

Lesson 2: Basics of CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to add colors, fonts, and responsive designs.

  • Selector: Targets HTML elements.
  • Property: Defines the style (e.g., color, font-size).
  • Value: Specifies the property setting (e.g., red, 16px).

CSS makes websites visually appealing. Experiment with different styles to see the impact!
